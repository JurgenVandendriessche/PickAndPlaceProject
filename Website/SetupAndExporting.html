Setting up reels, exporting files, saving and loading the project

Reel setup

After configuring the footprints you only need to do one more thing: give them a place on the machine.
Press the generate button and the program wil automatically make a first setup for the machines.

You can now change the position of the reels in the machine until the setup is like you wan't. When you lock a reel it will no longer show up for changing. The speed value is the speed for that specific reel. By default this speed wil be the same as the default speed of the machine.

WARNING: Changing the machine settings or certain settings of any footprint (Stack type or nozzle) will lead to a reset of the setup (and your progress).

Exporting

To export the files you can press on the Export button for one phase or you can press on the Export all button for all phases.
When you export one phase the name will be the same as your project name. If you export all phases at once the files will be named ProjectName_PhaseNumber (e.g. "Tutorial_Phase_1", "Tutorial_Phase_2",...).

WARNING:Existing files are overwritten.
